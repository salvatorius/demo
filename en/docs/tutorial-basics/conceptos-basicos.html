Conceptos Básicos de Python

Variables y tipos de datos​

En Python, una variable es un nombre que se utiliza para referirse a un valor. Para asignar un valor a una variable, se usa el operador "="

nombre = "Juan"
edad = 25
altura = 1.75

En Python, hay varios tipos de datos, como cadenas, enteros, flotantes, booleanos, listas, tuplas y diccionarios. Puedes verificar el tipo de una variable usando la función type().

nombre = "Juan"
edad = 25
es_mayor_de_edad = True
lista = [1, 2, 3, 4, 5]
tupla = (1, 2, 3, 4, 5)
diccionario = {"nombre": "Juan", "edad": 25}

Operadores aritméticos y de comparación​

Python tiene muchos operadores aritméticos y de comparación que se pueden usar para realizar cálculos y comparaciones en variables.

suma = 2 + 2
resta = 5 - 3
multiplicacion = 2 * 3
division = 6 / 3
modulo = 5 % 2
potencia = 2 ** 3
igualdad = 2 == 2
diferencia = 2 != 3
mayor_que = 5 > 3
mayor_o_igual_que = 5 >= 3
menor_que = 3 < 5
menor_o_igual_que = 3 <= 5

Estructuras de control de flujo​

Las estructuras de control de flujo son herramientas importantes para controlar el flujo de ejecución de un programa. En Python, hay varias estructuras de control de flujo, como la estructura if, la estructura for y la estructura while.

# estructura if
edad = 18
if edad >= 18:
    print("Eres mayor de edad")

# estructura for
numeros = [1, 2, 3, 4, 5]
for numero in numeros:
    print(numero)

# estructura while
contador = 0
while contador < 5:
    print(contador)
    contador += 1

Funciones​

Las funciones son bloques de código reutilizable que realizan una tarea específica. En Python, se pueden definir funciones usando la palabra clave def.

def suma(a, b):
    resultado = a + b
    return resultado

resultado = suma(2, 3)
print(resultado)

Módulos​

Los módulos son archivos de Python que contienen funciones y variables. Pueden ser importados a otros archivos para usar sus funciones y variables.

# importar el módulo math
import math

# usar la función sqrt del módulo math
resultado = math.sqrt(4)
print(resultado)

Consejos prácticos​

  • Es importante practicar la programación en Python regularmente para consolidar los conceptos.
  • La documentación oficial de Python es una excelente fuente de información y se puede encontrar en la página oficial de Python.
  • Hay muchos recursos en línea gratuitos y de pago para aprender Python, como Codecademy, Coursera, y Udemy.
  • También es importante participar en la comunidad de Python para hacer preguntas y compartir conocimientos. Hay varios foros y comunidades en línea, como Stack Overflow y Reddit.
